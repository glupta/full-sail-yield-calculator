# Full Sail Yield Calculator

A yield projection tool for Full Sail Finance LPs and SAIL token buyers on Sui.

## Overview

This tool enables users to:
- **LPs**: Project concentrated liquidity yields, model IL, compare oSAIL strategies
- **Token Buyers**: Project veSAIL locking rewards, value positions for secondary sale
